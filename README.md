# BoardGameBoard
Group 5 Proposal
## Overview
BoardGameBoard is a social-media-like platform where users can organize their games, write reviews for them, and post them to their profile, where other users can see them and interact with them. Users can build a top-5 board game list, and connect with other users based on the kind of games they like. 
## Problem
Board game fans are overwhelmed with choice in the modern board gaming climate. This leads to disorganized board game collections, forgotten impressions about games played in the past, and inability to decide on a game to play with friends. Board Game Board will alley these issues using modern social media tools.

There is currently a popular board game website called Board Game Geek. At a surface level it offers very similar features. However, it does so in an extremely complex way that can be offputting to users looking for a simple organization tool. Reviews tend to be pages long, and while this can be useful it can also be a chore when you want a simple overview of the games. It also lacks common social media implementations such as a user feed and tag organized posts.
## Epics
### Personalized User Page
This page should include a top five list of board games, a user information card that displays the user's username, profile picture, and status, a section showing recent play sessions and reviews, and a mutual friends list that is visible when the viewer is logged in. 
### Board Game Details Page
Shows the title, description and detailed information such as player count, age recommendations, complexity ratings and aggregate review ratings. Also uses the Board Game Geek api to gather alternate review scores..
### Personal lists and favorites
Users can create a top-five board game list, favorite games, or custom lists such as “Want to Play,” “Played,” and “Owned.”
### Review Posting
Users can log a game they played, give it a rating, write a review, and see their review history. 
### Local Board Game Stores
Users can quickly and easily find a map of their local board game stores. We will be using IPStack and Google Maps apis to facilitate this.
### Feed
A constantly updating, infinity scrollable feed of reviews and play session overviews from your friends and from games that you have followed.
### Recommendations and discovery (Future Feature)
The app can recommend board games based on a user’s ratings, favorites, preferred categories, player count, or play history. 
### Session planning with friends (Future Feature)
Users can create planned board game sessions, invite friends, suggest games, and view session details.
